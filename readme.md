# Thousand Lines - 千际线

一个基于Spring Boot开发的漫画分享平台，支持用户注册、漫画上传、在线阅读、搜索等功能。

## 🌟 项目特色

- **用户系统**: 完整的注册、登录、个人资料管理
- **漫画管理**: 支持漫画上传、编辑、删除
- **在线阅读**: 流畅的漫画阅读体验
- **搜索功能**: 支持按标题、作者、标签搜索
- **响应式设计**: 适配各种设备屏幕
- **邀请码系统**: 支持用户注册邀请机制
- **头像上传**: 用户可以自定义头像

## 🛠️ 技术栈

- **后端**: Spring Boot 3.2.0, Spring Data JPA, Spring Security
- **数据库**: MySQL 8.0
- **模板引擎**: Thymeleaf
- **前端**: HTML5, CSS3, JavaScript, Font Awesome
- **构建工具**: Maven
- **Java版本**: JDK 21+

## 🚀 快速开始

### 环境要求
- JDK 21 或更高版本
- MySQL 8.0+
- Maven 3.6+

### 数据库配置
1. 创建数据库：
```sql
CREATE DATABASE thousand_lines CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
```

2. 修改配置文件 `src/main/resources/application.properties`：
```properties
spring.datasource.url=jdbc:mysql://localhost:3306/thousand_lines?allowPublicKeyRetrieval=true&useSSL=true&serverTimezone=UTC&characterEncoding=utf8
spring.datasource.username=root
spring.datasource.password=1122  # 修改为你的MySQL密码
```

### 构建与运行

1. 克隆项目：
```bash
git clone <项目地址>
cd ThousandLines
```

2. 编译项目：
```bash
./mvnw clean compile
```

3. 运行项目：
```bash
./mvnw spring-boot:run
```

4. 访问应用：
打开浏览器访问 `http://localhost:8080`

### 默认账户
- 管理员账户：需要在数据库中手动创建
- 普通用户：通过邀请码注册
